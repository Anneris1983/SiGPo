💰 Métricas Clave de Liquidez
A = Ingresos Proyectados − Egresos Ejecutados
—
Liquidez Potencial Inmediata
Máxima solvencia. Efectivo disponible si cobrás todo hoy y no pagás nada más.
B = Egresos Proyectados Pendientes
—
Deuda Pendiente (Pasivo Exigible)
Compromiso asumido aún no pagado. Dinero que ya no te pertenece (incluye saldos parciales).
C = (A) − (B) = Flujo de Caja Neto
—
Margen de Maniobra
Número real de planificación. Positivo = Ganancia/Reserva. Negativo = No alcanza para pagar.

2. Ahorro por Virtualización
MONITOREAR
(Pasajes + Movilidad + Agapé) / Egresos Proyectados Totales
28%
Interpretación: 28% del presupuesto es "lujo presencial". Virtualizando recuperás $656.000.
Acción: Si la liquidez se complica, activar modalidad virtual libera casi un tercio del presupuesto.
3. Dependencia de Mora (5%)
CRÍTICO
Intereses Cobrados / Total Ingresos Ejecutados
22%
Interpretación: 22% de tus ingresos vienen de multas, no de cuotas. El negocio depende de pagos tardíos.
⚠️ PELIGRO: El alumno que paga mucha mora termina dándose de baja (Churn). Mejorar incentivos de pago puntual.
4. Índice de Cobrabilidad Real
BUENO
Ingresos Ejecutados / Ingresos Proyectados Originales
87%
Interpretación: De lo que esperabas cobrar, estás recaudando el 87%. Es realista para LATAM.
Acción: Usar 85-90% como baseline para proyecciones futuras, no 100%.
