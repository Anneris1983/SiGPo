💰 Métricas Clave de Liquidez
A = Ingresos Proyectados − Ingresos Ejecutados
—
Cobros Pendientes
Ingresos esperados que aún no ingresaron.
B = Egresos Proyectados − Egresos Ejecutados
—
Deuda Pendiente (Pasivo Exigible)
Compromiso asumido aún no pagado.
C = Ingresos Proyectados − Egresos Ejecutados
—
Liquidez Potencial Inmediata
Efectivo disponible si cobrás todo hoy y no pagás nada más.
D = A − B
—
Flujo Pendiente Neto
Cobros pendientes menos compromisos pendientes.
E = C − B
—
Margen de Maniobra
Positivo = Reserva. Negativo = No alcanza para pagar.
