📥 Importar Cohorte desde Excel

📋 Formato del Excel

El archivo Excel debe tener las siguientes columnas:

Columnas obligatorias:
• Programa
• Cohorte
• DNI
• Apellido
• Nombre
• Email
• Descuento %
• Cantidad Cuotas

Por cada cuota (C1, C2, C3...):
• C1_Monto
• C1_Estado (Abonada/Pendiente/Mora/Parcial/vacío)
• C1_Fecha_Pago (DD/MM/AAAA)
• C1_Abonado (monto pagado, solo si es parcial)
...repetir para C2, C3, etc.

⚠️ Importante:

  • La columna "Cantidad Cuotas" define cuántas cuotas debe el estudiante
  • Las fechas de pago son obligatorias si Estado = "Abonada" o "Parcial"
  • El formato de fecha debe ser DD/MM/AAAA o AAAA-MM-DD
  • Si Estado = "Parcial", debe completar C#_Abonado con el monto pagado
