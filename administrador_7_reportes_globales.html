⚠️ Índices Críticos de Cobertura
1. Cobertura de Gastos Críticos
—
Ingresos Ejecutados / Egresos Proyectados CRÍTICOS
—
Interpretación: La plata que ya entró alcanza para pagar X veces las plataformas e IT. ✅ >1 = Solvente.
Acción: Ninguna si >1. Si <1, priorizar gastos críticos.
2. Ahorro por Virtualización
—
(Pasajes + Movilidad + Agapé) / Egresos Proyectados Totales
—
Interpretación: % del presupuesto que se libera virtualizando clases presenciales.
Acción: Si la liquidez se complica, activar modalidad virtual.
3. Tasa de Mora
—
Monto en Mora / Ingresos Proyectados Totales
—
Interpretación: % de ingresos que no se cobró a tiempo.
⚠️ Alerta: >15% indica problema sistémico de cobranza.
4. Índice de Cobrabilidad Real
—
Ingresos Ejecutados / Ingresos Proyectados
—
Interpretación: De lo que esperabas cobrar, ¿qué porcentaje entró?
Acción: Usar este % como baseline para proyecciones futuras, no 100%.
